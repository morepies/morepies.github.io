Quick User Guide

5-minute onboarding flow (for users and reviewers)

1. Install & Sign In

Install the extension, set backend URL, then use Google login and token verification before running tasks.

2. Prepare Assets

Add source assets (images/videos), confirm assets are loaded, then run script parsing.

3. Parse Script

Use Parse to split script into shots; verify shot count, duration, and asset bindings.

4. Set Parameters

Choose aspect ratio, output count, and download mode; image mode can use first-frame confirmation.

5. Run Automation

Click Start to run generation and download workflow; use logs to quickly locate failed steps.

6. Verify Outputs

Check naming, quality, and quantity after completion; re-run failed shots when needed.

Note: This guide describes Flow plugin automation workflow, not the native quota or rights of the underlying model platform.
