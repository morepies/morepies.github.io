Product Overview

Chrome extension for batch operations and download optimization in Flow workflows

The Flow plugin focuses on input management, script queue processing, auto download, and asset handling in generation workflows.

Note: This is an independent tool service and is not affiliated with Google or its brands.

Core Features

Suitable for creators, studios, and asset-heavy pipelines

Script Queue

Batch execution, progress tracking, and resume support to reduce repetitive work.

Video Download Manager

Auto download, naming strategy, retry logic, and duplicate filtering.

Image Reference Mode

Supports reference-image workflows and cleanup policies for different usage habits.

Bilingual Interface

Chinese/English interface support for multilingual collaboration.

Parameter Control

Aspect ratio, output count, model options, and download strategy control.
